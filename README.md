# 🛒 Shop Data Analysis Project

## 📌 Project Overview
This project is a data analysis system for an online shop. It uses Python, MySQL, Pandas, NumPy, and Matplotlib to analyze sales data and generate business insights.

---

## ⚙️ Tools & Technologies Used
- Python
- MySQL
- Pandas
- NumPy
- Matplotlib

---

## 📊 Features
- MySQL database connection
- Data extraction using SQL JOIN
- Data cleaning (missing values, duplicates)
- Feature engineering (total_amount)
- Revenue calculation
- City-wise sales analysis
- Product-wise and category-wise analysis
- Customer insights
- Data visualization using charts

---

## 📈 Key Analysis
- Total revenue calculation
- Average order value
- Maximum order value
- Top customers
- Gender-wise spending
- City-wise revenue distribution

---

## 📊 Visualizations
- Bar chart: City-wise revenue
- Pie chart: Category distribution
- Bar chart: Top customers
